# Go CRUD

A CRUD example written in Go.
This example uses MySQL for the database server.

## Usage

```
Import the go-crud-posts.postman_collection.json file into Postman.
```

```bash
cd go-crud/
```

```bash
CompileDaemon -command="./go-crud"
```
